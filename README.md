# 商品管理システム（防災用品の紹介サイト）

## 概要

このシステムは、店舗で扱う商品を閲覧することができます。

商品の新規登録、検索表示、削除を行うことができます。

## 主な機能

* ログイン・ログアウト機能
* 商品一覧、検索機能
* 商品新規登録、削除機能

## 開発環境

* PHP 8.2.4
* MYSQL 8.0.35
* Laravel 10.13.5
